# Bash-databaseEngine
basic bash database engine to create, insert, list and drop databases and also create, list, update, delete, insert, drop and select Table are supported

# Run steps:
```bash
chmod +x CLIMenu.sh
./CLIMenu.sh
```

# Features
1. using `csv` files to save the data
2. menu to list and choose that needs no prior experience to deal with it
3. sourcing different .sh files for easy readability
